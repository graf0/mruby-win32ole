mruby-win32ole
==============

A port of CRuby's WIN32OLE to mruby.

## Installation

In the mruby `build_config.rb` file add:

```ruby
conf.gem :git => 'git://github.com/sdottaka/mruby-win32ole.git'
```

## Examples

See `mruby-win32ole/example` directory.
